What is SIMFIC Search Engine ?

A QBE book retrieval system built on hand-crafted features extracted from English and German Fiction Books.System supports users with global textual explanation for top-k items in the ranked list. Explanation tells the user what are the important features that are probably responsible.
For an avid reader, novels are a treat and the main dilemma the reader faces is selecting the next book to read once his present one is completed. The aim of this paper is to solve this dilemma by coming up with SIMFIC 2.0 (Similarity in Fiction 2.0), a query by example search engine which retrieves similar books given a query book.
